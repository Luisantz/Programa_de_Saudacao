# Programa de Saudação em Java

Este projeto é um programa simples em Java que interage com o usuário via terminal.  
Ele solicita o nome e a idade do usuário e, em seguida, exibe uma mensagem personalizada.

## 📋 Funcionalidades
- Solicita o nome do usuário.
- Solicita a idade do usuário.
- Exibe uma mensagem de saudação com nome e idade.
 
## 🖥️ Exemplo de uso

Olá qual o seu nome?

Luis

Olá qual a sua idade?

25

OLá Luis você tem 25 anos!

## 📂 Estrutura do código
- Scanner é utilizado para capturar entradas do usuário.
- System.out.println exibe mensagens no console.
- System.out.printf formata a saída final com nome e idade.

## 🔧 Requisitos
- Java 11 ou superior.
- Console/terminal para execução.

Feito para fins de aprendizado e prática de entrada/saída em Java.
